                             INSTRUCTIONS
========================================================================

Welcome to Grove Guardian, an exciting adventure where you must protect the last grove from impending threats. Follow the instructions below to start playing:

1. **Installation:**
   - Ensure you have GameMaker Studio 2 installed on your machine.
   - Clone the repository: `git clone git@github.com:Sm3ggl3s/GroveGuardian.git`
   - Open the project in GameMaker Studio 2.
   - Run the game from GameMaker Studio 2.

2. **Gameplay:**
   - Use WASD to move your character.
   - LMB to shoot
   - RMB to place towers
   - P to switch between views

3. **Cheats (Optional):**
   - Space + 0 is an auto win
   - Space + 1 adds money (50 each time)
   - Space + 2 regains health
   - Adds tower to your inventory:
    - Space + 4 adds Pond
    - Space + 5 adds Pellet Plant
    - Space + 6 adds Banana Split
    - Space + 7 adds Fervor Flower
    - Space + 8 adds Shackle Vines
    - Space + 9 adds fertilizer

4. **Enjoy the Game!**
   - Immerse yourself in the world that is the Grove Guardians.
